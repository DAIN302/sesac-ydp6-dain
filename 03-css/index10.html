CSS display 속성에 대해 알아보기

  • display 속성은 화면에 어떻게 보여질지 결정하는 속성
  • html 태그는 기본적으로 block 과 inline 요소 둘 중 하나를 기본값으로 가짐

inline

  1. span, a, img 태그와 같이 내용물(content) 만큼만 공간 할당
  2. content만큼 공간 핳당하기 떄문에 width, height, padding, margin 사용 불가
  3. 줄바꿈 없이 바로 옆에 배치
div-inline1 display 속성이 inline
div-inline2

block

  1. div, p, ul, ol 태그와 같이 한 줄 다 차지
  2. width, height, padding, margin 사용 가능
div-block1 display 속성이 block
div-block2

inline-block

  1. block 요소처럼 크기 지정을 하되(width, height, padding, margin 사용 가능)
  2. inline 요소처럼 다른 내용들과 같은 줄을 공유하고 싶은 경우
  3. block요소와 inline 요소의 장점을 가짐
